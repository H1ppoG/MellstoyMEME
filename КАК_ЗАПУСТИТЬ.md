# Как запустить Меллстрой мемы (Node.js)

## 1. Скачать и установить Node.js

1. Зайди на сайт: **https://nodejs.org**
2. Скачай **LTS**-версию (кнопка зелёная, например "22.x.x LTS").
3. Запусти установщик и нажимай "Next", ничего не меняя (можно оставить галочки по умолчанию).
4. В конце нажми "Finish".
5. **Перезапусти компьютер** или закрой и заново открой терминал/VS Code, чтобы подхватился Node.

**Проверка:** открой терминал (PowerShell или cmd) и введи:
```bash
node -v
npm -v
```
Должны появиться номера версий (например `v22.11.0` и `10.2.4`). Если команды не найдены — Node не установился или терминал не перезапускали.

**Если в PowerShell пишет "running scripts is disabled":** выполни один раз:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
Подтверди `Y` и Enter. После этого снова запусти `npm install`. Либо открой **cmd** (не PowerShell) и там выполни `npm install` — в cmd эта ошибка не появляется.

---

## 2. Открыть папку проекта в терминале

1. Открой папку проекта: `C:\Users\idontcareaboutsleep\Desktop\MemeSite`
2. В этой папке открой терминал:
   - в **VS Code / Cursor**: меню **Terminal → New Terminal** (или Ctrl+`);
   - либо в проводнике в адресной строке напиши `cmd` и Enter — откроется cmd в этой папке.

Убедись, что в терминале путь именно к папке MemeSite (должно быть что-то вроде `...\Desktop\MemeSite>`).

---

## 3. Установить зависимости

В терминале выполни одну команду:

```bash
npm install
```

Подожди, пока всё установится (появится папка `node_modules`). Ошибок быть не должно.

---

## 4. Запустить сервер

В том же терминале выполни:

```bash
npm start
```

или:

```bash
node server.js
```

Должно появиться сообщение вроде:
```
Server listening on http://localhost:3000
```

**Окно терминала не закрывай** — пока оно открыто, сайт работает.

---

## 5. Открыть сайт в браузере

- Открой браузер и перейди по адресу: **http://localhost:3000**

Ты увидишь главную страницу с мемами. Форма «Предложить свой мем» и поиск работают.

**Админка:** чтобы одобрять заявки, открой **http://localhost:3000/admin.html**  
Используй админ-ключ (по умолчанию в коде: `change-me-admin-key` — лучше поменять в `server.js` или через переменную окружения).

---

## 6. Дать доступ друзьям (одна сеть Wi‑Fi)

1. Сервер должен быть запущен (`npm start` или `node server.js`).
2. Узнай свой IP:
   - в терминале введи: `ipconfig`
   - найди блок «Адаптер беспроводной локальной сети» (Wi‑Fi) и строку **IPv4-адрес**, например: `192.168.1.5`
3. Друзьям скинь ссылку: **http://192.168.1.5:3000** (подставь свой IP). Они должны быть в той же Wi‑Fi сети.
